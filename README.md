# Brain Games

5 mathematical games to warm-up your mind skills!

## System requirements

- Linux family OS or MacOS;
- Node.js v16 or newer;

## Setup
````git clone https://github.com/timofeevmg/frontend-project-lvl1.git````

````cd frontend-project-lvl1````

````make install````
